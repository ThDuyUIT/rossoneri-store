# ROSSONERI STORE

# 1 - Introduction
Rossoneri store is an Ecommerce application  which can run on Android and Chrome

# 2 - Technologies:
- Framework: Flutter
- State management: Riverpod
- Database: Firestore

# 3 - Features:
  * Client:
    - Sign in/Sign up/Reset password
    - Manage cart
    - Create order
    - Logic checkout
    - Leave review
  * Admin: (Updating)
